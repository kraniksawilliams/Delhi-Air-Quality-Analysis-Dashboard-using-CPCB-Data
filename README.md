

# Delhi Air Quality Analysis Dashboard using CPCB Data

## Project Overview
An interactive Power BI dashboard built using CPCB air quality data to analyze pollution trends, AQI patterns, pollutant concentrations, and environmental insights across Delhi.

## Objectives
- Monitor Air Quality Index (AQI)
- Identify pollution trends
- Analyze pollutant concentrations
- Compare air quality across locations
- Support data-driven environmental decisions

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX
- CPCB Dataset

## Dashboard Features
- AQI Overview
- Pollutant Analysis
- Time Series Trends
- Interactive Filters
- KPI Cards
- Drill-through Reports

## Dataset
Source: Central Pollution Control Board (CPCB) India. CPCB provides air-quality monitoring data used for AQI analysis. :contentReference[oaicite:0]{index=0}

## Key Insights
- PM2.5 and PM10 are major contributors to poor AQI.
- Seasonal variations significantly affect pollution levels.
- Certain locations consistently report higher AQI values.
